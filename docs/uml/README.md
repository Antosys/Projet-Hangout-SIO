# Hangout - UML

L'UML a été généré grâce à plant UML, une application capable de reproduire en UML les bases de données et systèmes complexes.

Le fichier docs/uml/diagramme-de-classes-uml.puml vous permet de voir comment celui-ci a été construit. En effet, grâce à la documentation intuitive de plant uml, j'ai simplement dû recopier les éléments de mon infrastructure dans le format que plant UML attendait. Voir https://plantuml.com/fr/class-diagram
